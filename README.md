# NASDAQ-100 200日均线宽度指标

计算 NASDAQ-100 成分股的 200日均线宽度指标（% of stocks above 200-day MA）

## 指标定义

```
Breadth200(t) = Count[ Close(i,t) > SMA200(i,t) ] / 有效股票数 × 100
```

## 安装

```bash
pip install yfinance pandas pytest
```

## 运行

```bash
# 最新数据
python src/ndx_breadth.py

# 指定日期
python src/ndx_breadth.py 2026-03-15
```

## 运行测试

```bash
pytest tests/ -v
```

## 数据来源

- **原型**: yfinance (Yahoo Finance)
- **数据**: 过去2年历史数据（确保200+交易日）
